Discover all existing tests (unit + functional) that cover the code being modified in this epic.

## Objectives

1. Use aap_dev_discover_tests to run structural test discovery
2. Review AGENTS.md for test locations and conventions
3. If tests live in a separate repo, include that repo's test coverage

## Expected Output

Produce a test coverage map in beads issue notes:
- Which source files have test coverage
- Which test files cover each source file
- Test names and categories (unit vs functional)
- Coverage gaps: source files with no tests

## Tools
- aap_dev_discover_tests
- Read AGENTS.md for test repo locations
